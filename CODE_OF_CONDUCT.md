# Code of Conduct

## Our Commitment

We are committed to a respectful, harassment-free community experience for everyone.

## Expected Behavior

- Be respectful and professional.
- Assume good intent and discuss ideas constructively.
- Give actionable technical feedback.
- Focus on project goals and user safety.

## Unacceptable Behavior

- Harassment, discrimination, or hateful conduct.
- Personal attacks, intimidation, or sustained disruption.
- Publishing private information without permission.
- Trolling, baiting, or bad-faith participation.

## Scope

This Code of Conduct applies in:

- repository issues and pull requests,
- discussions and review threads,
- project-related community spaces.

## Reporting

If you experience or witness unacceptable behavior, report it privately to project maintainers.
For security issues, use `SECURITY.md` and private reporting channels.

## Enforcement

Maintainers may take any action they deem appropriate, including warning, temporary restriction,
or removal from project spaces for violations.
